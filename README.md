## Über dieses Projekt

Dieses Repository enthält die Verkaufsseite für das E-Book **"Der Senior Hund Ratgeber"** — ein persönlicher Erfahrungsbericht aus über 17 Jahren mit zwei alten Hündinnen.

**Chippa** ist 17 Jahre alt, ein kleiner Mischling — unzerstörbar, neugierig, und schläft am liebsten auf Einkaufstaschen.

**Jackie** ist 16 Jahre alt, eine Berger-Mischlingsdame — weise, still, und fast blind. Mit einer Würde, die man sich von Menschen wünschen würde.

Dieses Buch ist für alle, die wissen wovon ich spreche. 🐾

## Deployment

Die Seite läuft auf **GitHub Pages** — kostenlos, ohne Server.

### Aktivieren

1. Repository auf GitHub erstellen
2. `index.html` hochladen
3. Einstellungen → Pages → Branch: `main` → Ordner: `/ (root)`
4. Fertig — die Seite ist erreichbar unter:

```
https://DEIN-USERNAME.github.io/herzpfote24/
```

Diese URL trägst du bei **Digistore24** als Verkaufsseite ein.
